Welcome to Mini Formatter Extension!

Thank you for installing the Mini Formatter Extension for Chrome! This extension is designed to simplify code formatting directly in your browser. Whether you're working with HTML, CSS, or JavaScript, Mini Formatter makes it easy to beautify and clean your code with a simple click.

Features

  • Code Formatting: Automatically formats HTML, CSS, and JavaScript for cleaner and more readable code.
  • File Upload: Load code from local files in supported formats and instantly format them within the extension.
  • Copy to Clipboard: Quickly copy the formatted code to your clipboard.
  • Download Formatted Code: Save your formatted code as a text file for easy sharing or backup.
  • Customizable Theme: Switch between light and dark modes to enhance your coding experience.
